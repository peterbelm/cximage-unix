# cximage-unix

Computer vision library for CPP.

# Building

To build as a shared library use scons:

```shell
scons
```

To clean:

```shell
scons -c
```

To install to /usr/local/lib (you may need to be root):

```shell
scons install
```

To uninstall:

```shell
scons -c install
```
